==============
Installation
==============

.. start installation

.. tabs::

	.. tab:: from PyPI

		.. prompt:: bash

			python3 -m pip install domdf_sphinx_theme --user


	.. tab:: from GitHub

		.. prompt:: bash

			python3 -m pip install git+https://github.com/domdfcoding/domdf_sphinx_theme@master --user

.. end installation

Then in your ``conf.py`` file:

.. code-block:: python

	html_theme = "domdf_sphinx_theme"
